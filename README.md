## Quick Installation

```sh
sh -c "$(curl -sSL dots.spywhere.me)"
```

## Installation with Additional Flags and Options

To use flags in remote installation, use this command

```sh
sh -c "$(curl -sSL dots.spywhere.me)" - [flags...]
```
